Note: it requires yarn installed 

> Initial steps
```bash
$ git clone https://github.com/ruifernandees/proffy.git

```

> Run back-end (API REST)
```bash

# Access the back-end folder
$ cd server

# Install all dependencies
$ yarn install

# Run all migrations
$ yarn knex:migrate

# Start the server
$ yarn start

```
> Run web front-end
```bash
# Access the web front-end folder
$ cd web

# Install all dependencies
$ yarn install

# Run the web project
$ yarn start
```
